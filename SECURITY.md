# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in AgentEval CLI, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email: **agenteval@joslat.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest pre-release | ✅ |
| Older versions | ❌ |

## Scope

This policy covers the AgentEval CLI tool itself. For vulnerabilities in the core AgentEval library, please report to the [main AgentEval repository](https://github.com/AgentEvalHQ/AgentEval).
